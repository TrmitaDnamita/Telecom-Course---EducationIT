HTML y CSS

HTML es el lenguaje de marcado que permite estructurar el contenido de una página web mediante etiquetas.

Mientras que CSS se utiliza para aplicar estilos visuales a los elementos HTML, como colores, fuentes y márgenes.

Nota: si bien es posible incorporar CSS dentro del HTML, no es recomendable, ya que HTML no debería usarse con fines de estilo, sino estructurales.

Etiquetas de HTML más comunes:

  • Los encabezados, que van desde h1 hasta h6.
  • Los elementos semánticos como article, section, header y main, que permiten organizar mejor la estructura de la página web.
  • Las listas, que pueden ser tanto numeradas como no numeradas, para una mejor lectura de características.
  • Y otras etiquetas igual de importantes que puedes encontrar en: Etiquetas HTML.

Propiedades básicas de CSS:

  • margin permite modificar los márgenes de tu página o sus etiquetas. Se puede usar de forma general o especificar el lado que se desea modificar (left, right, top, bottom).
  • padding es otra propiedad similar a margin, con la diferencia de que establece la distancia interna entre el contenido y los bordes de una etiqueta. También puede aplicarse de forma general o específica.
  • font te ayuda a modificar el estilo de los textos con una variedad de opciones como tamaño, tipo, alineación, estilo, etc.
  • Para más información, puedes consultar todas las propiedades en: Propiedades CSS.
